CSS Specificity

CSS has specificity and cascading algorithm that helps to distinguish between selectors ... it may be btwn id and class or class vs class or element vs class or inline vs class. And here there is an rule that is 
" Inline CSS (1000) > ID selector (100) > Class or attribute selector (10) > Element Selector (1) > Universal Selector (0) "

* We could also bypass inline selector using the "important" flag in css. Pseudo class have 10 specificity !!

I am given !important style that overrides the inline style also !!
